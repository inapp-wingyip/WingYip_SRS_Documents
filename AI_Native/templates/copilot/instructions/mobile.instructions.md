## Mobile Conventions

- Android-only HHD app (React Native 0.72).
- Use feature-based navigation with React Navigation.
- Implement offline support with local storage sync.
- Use Zustand for global state.
- Test on actual HHD hardware before release.
- Run `npm run lint && npm test` before committing.
